# Remarxks Mechanical Spawner

A Create mod addon that adds a **Mechanical Spawner** — a kinetic-powered block that spawns mobs using rotational force.

## Features

- **Stress-powered mob spawning** — the Mechanical Spawner requires rotational force (SU) to operate. Higher RPM increases spawn rate and ejects entities with greater velocity.
- **Configurable entity type** — the spawner can be bound to any mob via two methods: sneak-right-clicking a vanilla spawner with the item, or right-clicking the placed block with a spawn egg.
- **Configurable spawn limits** — two server-side settings control the radius for checking nearby entities and the maximum number of entities allowed before spawning pauses.
- **Custom rendering** — the bound entity is visually displayed spinning inside the block, and also rendered on the item when held in hand.
- **Ponder integration** — interactive in-game guide scenes explain the block's mechanics.
- **Flywheel support** — optimized rendering via Create's Flywheel engine.
- **Data preservation** — the block drops itself with its bound entity type preserved when mined.
- **Custom crafting recipe** — assembled in the Mechanical Crafting Table using Create components.
